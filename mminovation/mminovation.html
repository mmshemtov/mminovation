My Coding Courses

Codecademy: Basic Java

This was a basic java course learning everything between building a basic java sheet and making varisbles, to writing for loops and functions. It also taught me how to write arrays and 2d arrays. It was a 25 hour course which i think takes a little shorter to complete. I've forgotten most of it but I don't think it would take me long to freshen my skills.

Team Treehouse: Web-development

This course teaches you the necessary html, css, and javascript to make a responsive website with nice design and some features. It's a different style than codecademy was since this is a bunch of videos with someone showing you step by step what to do, vs codecademy you actually had to do each step on your own. It's a 56 hour course though it took a while for me to complete it and I still have to review a lot of it.
